# Celebrity Selfie Video

Composites a user photo with a specified celebrity-style character into a shared frame. Core: uses img2img to preserve the user's real appearance, generating a celebrity character naturally sharing the same scene, with strong realistic feel.

> Note: Generated content is AI-created imagery. The celebrity likeness is generated by the model based on public information and is for entertainment purposes only.

## Key Principles

- **User photo is the img2img reference**: user's appearance, hairstyle, and clothing must be preserved in the generated image
- **Specify celebrity via character description**: describe the target celebrity's appearance features in detail (hair color, face shape, signature features) — not just by name
- **Scene setting must be natural and realistic**: both in a reasonable life setting (street shot, café, event venue etc.) — no PS-composite feel
- **Interaction must be natural**: the two have a reasonable visual relationship or subtle interaction, not both staring at the camera

## Workflow

### 1. Confirm Celebrity and Scene

**Required info**:
- User photo (upload to get asset_id)
- Target celebrity name
- Scene style (Polaroid selfie / street shot / event red carpet / indoor scene etc.)

**Celebrity appearance description extraction**: based on the celebrity name, extract their signature appearance features (hairstyle color, facial features, typical outfit style), and write into the prompt. Don't just write the name — the platform relies on appearance descriptions, not celebrity name indexing.

### 2. Upload User Photo, Send Shot List

```
Art style: realistic feel, real photo texture, not commercial studio style

Shot 1:
- Start frame: reference original image [user photo asset_id], preserve user's appearance, hairstyle, and clothing features
- Scene: [specific scene description, e.g.: city street, natural light, afternoon]
- Celebrity character: [celebrity's signature appearance feature description: hairstyle/hair color/facial features/outfit style]
- Interaction: [both standing shoulder to shoulder / celebrity arm around user's shoulder / both smiling at each other / holding up phone for selfie]
- Composition: selfie angle (slightly above eye level) or third-person shot, both clearly in frame
- Duration: [3–5s]
- Video description (if animated): both with subtle movement, natural breathing feel, background with light depth-of-field blur
```

**Scene reference**:

| Scene Type | Description Points |
|---|---|
| Polaroid selfie | Polaroid frame border visible in image, white edges, grainy texture, vintage tone |
| Street shot | City background, natural light, blurred pedestrians, strong realistic feel |
| Event red carpet | Red carpet background board, flash-light feel, formal attire |
| Café / indoor | Warm indoor light, table props, casual atmosphere |
| Concert / live event | Stage lighting background, blurred crowd, dynamic feel |

### 3. Check Output

| Check Item | Standard |
|---|---|
| User appearance preserved | User's hairstyle and facial contour match the original photo |
| Celebrity recognizability | Celebrity character matches the described signature features |
| Scene plausibility | Both in the same realistic scene, no obvious composite artifacts |
| Natural interaction | Both have a reasonable visual relationship or physical contact |

If user appearance is distorted: increase img2img similarity, add `preserve user's facial features exactly` to the prompt.
