# Brand Atmosphere Ad

Turns a product/brand and emotional direction into a high-end brand atmosphere film. Core: emotional arc, visual world, and fast-slow mixed-cut rhythm — not product feature demonstration. Reference: Nike, Red Bull, Apple brand films.

## Key Principles

- **Build the visual world first, then think about shots** — the one-sentence perception description drives everything
- **Lighting description = emotion description** — get the lighting right and image quality improves directly
- **Write atmosphere, not function** — always ask "what will the audience feel," not "what does this shot explain"
- **Fast-slow mixed cuts depend on duration planning** — decide which shots are fast (1–1.5s) and slow (2.5–4s) in Step 1
- **Color world consistency** — all shots must share the same color universe; isolated shots shatter the feel
- **Product integration must be natural** — the product is part of this world, not the object being displayed
- **Product must be preceded by emotional build-up** — in the shot before the product, the character must be in some "lacking/at the limit" state so the product's appearance feels earned
- **Write specific visible details for characters, not emotional labels** — not "sharp gaze," but "pupils slightly contracted, eyes fixed on a point outside the frame, faint vertical crease between the brows"
- **No tail frames**; **audio before video**

## Workflow

### 1. Analyze Product + Output Creative Concept and Storyboard

Run 2–3 WebSearch queries before storyboarding to find real-world creative references:

| Category | Search Terms |
|----------|-------------|
| Sports/energy drinks | `"Gatorade" OR "Red Bull" brand film storyboard creative` |
| Luxury/beauty | `"Chanel" OR "Dior" brand film storyboard script` |
| Sports brands | `"Nike" OR "Adidas" brand film shot list creative concept` |
| Lifestyle | `[brand name] brand film treatment creative brief` |

Extract: emotional arc patterns, camera language preferences, visual style benchmarks, clichés to avoid. Do not show this to the user — apply it directly to the storyboard.

**Establish four creative elements** — if the user hasn't stated them, determine them yourself based on product/category; do not ask:

- **Brand emotional keywords** (2–3): Power / Speed / Freedom / Solitude / Luxury / Purity / Wildness / Calm
- **Visual world direction**: one-sentence perception reference (e.g., "lonely sense of speed in the midnight city")
- **Narrative sense** — choose one:
  - Implicit story (character emotional arc) → sports/lifestyle
  - Pure rhythm (shots follow music, aesthetics driven) → fashion/music
  - Atmosphere stacking (montage of scenes/textures under one emotion) → luxury/beauty
- **Product appearance mode**:
  - Throughout — product appears naturally every few shots as a prop
  - Grand finale — pure atmosphere first, product/logo only in the last shots
  - Minimal presence — one fleeting shot; brand feel carried by color and style

**Emotional arc** — choose one:

| Arc | Description | Category |
|-----|-------------|----------|
| Suppression → Explosion | Restrained opening, rapid escalation, explosive climax | Sports/energy/racing |
| Quiet → Stunning | Extremely quiet beauty builds to a stunning image | Luxury/high-end beauty |
| Rhythmic Loop | No arc, pure visual rhythm | Music/fashion/art |
| Everyday → Transcendence | Ordinary moment elevates to transcendence | Lifestyle/mass |
| Conflict → Resolution | Tension between opposites (speed vs. stillness) | Auto/sports |

**Storyboard** (8–15 shots, 20–40s, 1–4s each):

| Shot Type | Emotional Function |
|-----------|-------------------|
| Extreme close-up texture (material/liquid/skin) | Establish perception, create tension |
| Slow-motion action moment | Amplify impact, create drama |
| Establishing shot | Build the world, set emotional tone |
| Character emotional moment (gaze/posture) | Emotional resonance |
| Product natural entry (as a prop, not showcased) | Natural brand integration |
| Abstract visual (light/geometry/motion blur) | Rhythm, decoration |
| Brand/product freeze frame (clean background) | Brand impression, closing |

Fast-slow rhythm: opening 1–2 shots slow (1.5–3s) → mid-section fast cuts (0.5–1.5s) on the beat → climax/product slow down (2–4s) → closing freeze (2–3s).

**Product placement rules:**
- First 50%: product at most once, as a background prop
- Middle 30%: product as a natural prop (held, casually placed)
- Last 20%: product can take the lead (brand closing)
- **Forbidden**: "trigger point" mode (character is thirsty → product satisfies need) — that's functional ad logic

**Self-review checklist** — fix issues before showing to user:

| Check | Criteria |
|-------|---------|
| Product appearance timing | Product appears as "part of this person's world," not as an answer to a need |
| Character continuity | Single protagonist throughout; multiple characters need a clear visual relationship |
| Adjacent shot logic | Each shot has a narrative reason to follow the previous one, not just "same tone" |
| Homogeneous stacking | No 3+ consecutive shots expressing the same emotion |
| Post-climax breathing | One descending/breathing shot after the emotional peak before the product or close |
| Opening rhythm | First 2 shots include at least one slow shot to establish the world |
| Character description depth | Character shots have specific visible physical details, not emotional labels |

Show creative concept to user, then proceed to Step 2 immediately.

### 2. Upload Product Images & Send Script

Upload product reference images only if product shots are needed; skip if pure atmosphere film.

Send one message with: global style + color tone + BGM description + character/scene definitions + all shots.

**Shot description principle — write atmosphere, not function:**
- Describe what the audience will feel, not what the shot explains
- Replace emotional labels with specific visible bodily signals:

| Emotional Label (avoid) | Specific Visible Detail (use instead) |
|-------------------------|--------------------------------------|
| "Gaze sharp as a blade" | "Pupils slightly contracted, eyes fixed on a point outside the frame, faint vertical crease between the brows" |
| "Expression heavy" | "Jaw lightly tightened, lips pressed flat, gaze angled 15° downward" |
| "Fist clenched with tension" | "Knuckle joints whitened, thumb pressing over index finger, faint muscle lines on forearm" |

**Lighting = emotion:**

| Emotion | Lighting |
|---------|---------|
| Suppression/tension | Single-side hard light, high-contrast shadows, cold white or blue-grey |
| Explosion/climax | Backlight bloom, intense edge light, near overexposure |
| Luxury/tranquility | Low-key soft top light, golden or warm white tones |
| Speed/dynamism | Motion blur, light streaks, high color contrast |
| Solitude/poetry | Diffused light, blue-grey low saturation, large empty space |

Each shot must be a single point-in-time cinematic composition — no dividing lines or multi-angle juxtaposition. No tail frames.

### 3. Filter Check

Check shots, dialogs, characters. Atmosphere film-specific:
- Shot durations vary — fast-slow rhythm present, not all the same length
- Voiceover (if any) is tagline level only — 5–10 words, strong imagery, does not explain the product
- BGM description is set

### 4. Clear All Tail Frames

Fast cuts require clean transitions — send a message to clear all tail frames, keep only start frames. Verify afterward.

### 5. Generate Audio

**BGM formula**: `[instrument texture] + [BPM/rhythm] + [emotional arc] + [beat point notes]`

| Brand Emotion | BGM Example |
|---------------|------------|
| Sports/explosion | Electronic synth base, BPM 130, drums explode at second 5, cold and aggressive, no vocals |
| Luxury/tranquility | Piano single note, strings slowly layer in, extremely slow, noble and restrained |
| Speed/freedom | Guitar riff + electronic beat, BPM 140, full of momentum |
| Tech/futuristic | Pure electronic synthesis, cold tone, low-frequency rumble, BPM 100 |
| Poetic/emotional | Cello solo, tempo rises and falls with emotion |

Voiceover is minimal or absent. If included: one tagline, 5–10 words, voice tone matches brand character.

### 6. Generate All Videos

After audio completes, generate all videos.

## Visual Language Reference

| Brand Emotion | Color World | Lighting | Composition | Rhythm |
|---------------|-------------|----------|-------------|--------|
| Power/explosion | Deep black + cold blue + red bursts | Single-side hard, high contrast | Low-angle upshot, dynamic tilt | Primarily fast, occasional slow-mo |
| Freedom/speed | Saturated blue-white, motion blur | Backlight, edge light | Wide shot, large empty space | Extreme fast + one or two slow |
| Luxury/restraint | Black-gold/off-white, low saturation | Soft diffused, precise spot | Minimal centered, large empty space | Overall slow, occasional fast |
| Tech/future | Dark + cold light | Glowing texture, blue-white bands | Symmetrical/geometric | Strong rhythm, mechanical feel |
| Nature/healing | Green-white warm beige | Natural diffused, no hard shadows | Breathing feel, off-center | Slow, gentle |
| Poetic/melancholic | Low-saturation blue-grey | Overcast diffused, no direct light | Large empty space, small figure | Extremely slow, long-shot feel |

## Resolution

Default: **9:16 vertical** (TikTok/Reels/vertical screens)

| Use Case | Resolution |
|----------|------------|
| TikTok / Reels / vertical screens | 720×1280 (9:16) |
| YouTube / horizontal ads | 1280×720 (16:9) |
| Square social media | 720×720 (1:1) |
